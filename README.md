## Mammmalian molar morphology is evolutionarily linked with dietary ecology
### Understanding this morphology can reveal unseen factors of the local ecosystem
* Tribosphenic molars consist of the following cusps:
1. Protoconid
2. Paraconid
3. Metaconid
4. Hypoconid
5. Entoconid
   
   *Another important aspect of mammalian molars is the _cristid obqliqua_

   * *The cristid obliqua to the medial margin length shows promising dietary inferences*
